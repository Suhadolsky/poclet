# PocketGrid - Project Plan

## 1. Identity
- **Name:** PocketGrid
- **Tagline:** Все о мире портативного гейминга
- **Stack:** Astro (Static Site Generator) + Tailwind CSS (Styling)
- **Host:** Netlify / Vercel (Fast, free tier, easy deploy)

- **Guides:** How-tos, emulation setups, mods.
- **News:** Quick industry updates.
- **Ads:** Placeholder slots (Header, Sidebar, In-content).

- Keyword-focused content (I will help write this).
